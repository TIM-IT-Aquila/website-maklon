# Instruksi untuk Claude Code

Dokumen ini sengaja tidak menduplikasi aturan proyek agar tetap menjadi single source of truth.

1. Baca [AGENTS.md](AGENTS.md) untuk konteks proyek, arsitektur, dan batasan implementasi.
2. Baca [CODE_STYLE.md](CODE_STYLE.md) untuk gaya kode, tanggung jawab layer, serta aturan DRY, SSOT, SoC, dan YAGNI.
3. Jalankan validasi yang tercantum di `CODE_STYLE.md` setelah melakukan perubahan.
